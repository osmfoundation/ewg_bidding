# OpenStreetMap Website: Copyright Page Update

The goal of this project is to update the [copyright page](https://www.openstreetmap.org/copyright) on the OSM website.

* Project: [openstreetmap-website](https://github.com/openstreetmap/openstreetmap-website)
* Related:
  - https://github.com/osmfoundation/ewg_bidding/issues/18

The expected deliverable is a mergeable pull request towards [openstreetmap-website](https://github.com/openstreetmap/openstreetmap-website). It must at least:
- Implement the copy changes described in [the CWG recomendations document](https://docs.google.com/document/d/1qthlWHa3-cQ9KcHai-TORAEnHD3LBI0vTi2OJ1TMz_k/edit?tab=t.0)
